# Java Tutorial Repository

A comprehensive collection of Java programming tutorials covering fundamental to advanced concepts. Each file contains well-commented code with detailed explanations and practical examples.

## 📚 Tutorial Contents

### Basic Concepts (01-05)
- **01_HelloWorld.java** - Introduction to Java syntax and structure
- **02_Variables.java** - Variable declaration, scope, and naming conventions
- **03_DataTypes.java** - Primitive and reference types, type casting
- **04_Operators.java** - Arithmetic, logical, relational, and bitwise operators
- **05_ControlFlow.java** - If-else, switch, loops, and jump statements

### Object-Oriented Programming (06-10)
- **06_Classes.java** - Classes, objects, constructors, and methods
- **07_Inheritance.java** - Extending classes, method overriding, super keyword
- **08_Polymorphism.java** - Method overloading/overriding, dynamic dispatch
- **09_Encapsulation.java** - Data hiding, getters/setters, access modifiers
- **10_Abstraction.java** - Abstract classes and methods

### Advanced Concepts (11-18)
- **11_Interfaces.java** - Interface implementation, default methods
- **12_ExceptionHandling.java** - Try-catch-finally, custom exceptions
- **13_Collections.java** - List, Set, Map, Queue operations
- **14_Generics.java** - Generic classes, methods, and bounded types
- **15_Streams.java** - Stream API, filter, map, reduce operations
- **16_FileIO.java** - File reading/writing, BufferedReader/Writer
- **17_Multithreading.java** - Thread creation, synchronization
- **18_Lambda.java** - Lambda expressions and functional interfaces

## 🚀 How to Use

### Compile a File
```bash
javac 01_HelloWorld.java
```

### Run the Program
```bash
java HelloWorld
```

### Compile and Run All Files
```bash
# Compile all
javac *.java

# Run specific file
java ClassName
```
